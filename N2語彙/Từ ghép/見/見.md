# 見 (みる) - Mind Map

見 「Nhìn → Nhận biết → Đánh giá → Hành động」

├── ① Nhìn (Visual Perception)
│   Trường nghĩa:
│   - Hành động nhìn hoặc kết quả nhìn thấy.
│
│   Phân biệt:
│   - [[見る]]      : Chủ động nhìn.
│   - 見える    : Tự nhiên nhìn thấy (không chủ ý).
│   - [[見せる]]    : Làm cho người khác nhìn.
│
├── ② Phát hiện (Discovery)
│   Trường nghĩa:
│   - Từ việc nhìn mà tìm ra, nhận ra điều trước đó chưa biết.
│
│   Phân biệt:
│   - [[見つける]]  : Tìm thấy vật/người cụ thể.
│   - 見出す    : Phát hiện ý tưởng, quy luật, giá trị.
	[[見出し]]: Tiêu đề/Đầu đề bản tin.
│   - 見付ける
│
├── ③ Quan sát (Observation)
│   Trường nghĩa:
│   - Không chỉ nhìn, mà còn theo dõi phạm vi hoặc quá trình.
│
│   Phân biệt:
│   - [[Chữa đề/Learning_Profile/CompoundVerb/見守る|見守る]]    : Dõi theo để bảo vệ, theo dõi.
│   - 見回す    : Nhìn xung quanh.
│   - 見渡す    : Nhìn bao quát toàn cảnh.
│   - 見続ける  : Tiếp tục nhìn trong thời gian dài.
│
├── ④ Kiểm tra / Xem lại (Review)
│   Trường nghĩa:
│   - Quan sát lại nhằm kiểm tra hoặc so sánh.
│
│   Phân biệt:
│   - [[Chữa đề/Learning_Profile/CompoundVerb/見直す|見直す]]    : Xem lại để sửa hoặc đánh giá lại.
│   - 見返す    : Lật xem lại; nhìn lại.
│   - 見比べる  : Đặt cạnh nhau để so sánh.
│
├── ⑤ Bỏ sót (Missing / Overlooking)
│   Trường nghĩa:
│   - Có nhìn nhưng không nhận ra hoặc cố tình bỏ qua.
│
│   Phân biệt:
│   - [[見逃す]]    : Bỏ lỡ cơ hội hoặc cảnh tượng.
│   - [[見落とす]]  : Nhìn sót chi tiết.
│   - 見過ごす  : Làm ngơ, bỏ qua.
│
├── ⑥ Đánh giá (Judgement)
│   Trường nghĩa:
│   - "Nhìn" theo nghĩa trừu tượng: nhận định, ước lượng.
│
│   Phân biệt:
│   - 見込む    : Dự đoán tiềm năng hoặc khả năng.
	[[見込み]]
│   - 見積もる  : Ước tính số lượng, giá cả.
	[[見積もり]]
│   - 見なす    : Coi như, xem là.
│
├── ⑦ Đối nhân xử thế (Human Interaction)
│   Trường nghĩa:
│   - Gặp gỡ hoặc hành động với người khác.
│
│   Phân biệt:
│   - 見舞う    : Thăm người bệnh.
│   - [[見送る]]    : Tiễn đi; hoặc quyết định không làm.
│
└── ⑧ Từ bỏ / Hoãn (Abandonment)
    Trường nghĩa:
    - Từ việc "quan sát" chuyển sang quyết định dừng hoặc từ bỏ.
    
    Phân biệt:
    - 見合わせる : Hoãn, tạm ngừng.
    - 見放す     : Bỏ mặc, không quan tâm nữa.
    - 見捨てる   : Ruồng bỏ hoàn toàn.
