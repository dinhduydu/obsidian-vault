# 手（て）

> **Core Meaning:** Tay → Hành động → Kỹ năng → Phương pháp → Can thiệp

手 「Tay → Hành động → Kỹ năng → Phương pháp → Can thiệp」

├── ① Dùng tay (Physical Action)
│   Trường nghĩa:
│   - Hành động trực tiếp bằng tay.
│
│   Phân biệt:
│   - 手に取る      : Cầm lên.
│   - 手渡す        : Trao tận tay.
│   - 手放す        : Buông tay, từ bỏ.
│   - 手招きする    : Vẫy tay.
│
│   Từ:
│   - [[手に取る]]
│   - [[手渡す]]
│   - [[手放す]]
│   - [[手招きする]]
│
├── ② Thao tác (Operation)
│   Trường nghĩa:
│   - Dùng tay để thao tác hoặc xử lý.
│
│   Phân biệt:
│   - 手掛ける      : Đích thân làm.
│   - 手伝う        : Giúp đỡ.
│   - 手入れする    : Bảo dưỡng, chăm sóc.
│
│   Từ:
│   - [[手掛ける]]
│   - [[手伝う]]
│   - [[手入れする]]
│
├── ③ Kỹ năng (Skill)
│   Trường nghĩa:
│   - Tay biểu thị năng lực hoặc trình độ.
│
│   Phân biệt:
│   - 手慣れる      : Thành thạo.
│   - 手こずる      : Vất vả vì đối tượng khó.
│
│   Từ:
│   - [[手慣れる]]
│   - [[手こずる]]
│
├── ④ Phương pháp (Method)
│   Trường nghĩa:
│   - "Tay" chuyển nghĩa thành cách làm hoặc biện pháp.
│
│   Phân biệt:
│   - 手を打つ      : Thực hiện biện pháp.
│   - 手を尽くす    : Làm mọi cách.
│   - 手立て        : Biện pháp, phương án.
│
│   Từ:
│   - [[手を打つ]]
│   - [[手を尽くす]]
│   - [[手立て]]
│
├── ⑤ Can thiệp (Intervention)
│   Trường nghĩa:
│   - Tay vươn tới đối tượng để tác động.
│
│   Phân biệt:
│   - 手を出す      : Ra tay; nhúng tay; động vào.
│   - 手を貸す      : Giúp sức.
│   - 手を加える    : Chỉnh sửa, bổ sung.
│
│   Từ:
│   - [[手を出す]]
│   - [[手を貸す]]
│   - [[手を加える]]
│
├── ⑥ Điều khiển (Control)
│   Trường nghĩa:
│   - Tay kiểm soát hoặc chi phối.
│
│   Phân biệt:
│   - 手に入れる    : Có được.
│   - 手に負えない  : Không xử lý nổi.
│   - 手中に収める  : Nắm trong tay.
│
│   Từ:
│   - [[手に入れる]]
│   - [[手に負えない]]
│   - [[手中に収める]]
│
└── ⑦ Từ bỏ / Rút lui (Withdrawal)
    Trường nghĩa:
    - Rút tay khỏi một việc hoặc từ bỏ.

    Phân biệt:
    - 手を引く      : Rút lui.
    - 手を切る      : Cắt đứt quan hệ.
    - 手放す        : Buông bỏ, từ bỏ quyền sở hữu.

    Từ:
    - [[手を引く]]
    - [[手を切る]]
    - [[手放す]]
