Tiền tố
 │ ├─ Phủ định / Đối lập
 │   ├─ 無 (Vô): Thiếu, không (Vd: 無意識 - vô thức, 無責任 - vô trách nhiệm)
 │   ├─ 非 (Phi): Không, phi (Vd: 非公開 - không công khai, 非合法 - phi pháp)
 │   ├─ 未 (Vị): Chưa (Vd: 未解決 - chưa giải quyết, 未完成 - chưa hoàn thành)
 │   ├─ 反 (Phản): Phản đối, đối lập (Vd: 反 Chính phủ, 反 tác dụng)
 │   └─ 逆 (Nghịch): Phản, ngược (Vd: 逆効果 - tác dụng ngược)
 │ ├─ Mức độ 
 │   ├─ 超 (Siêu): Siêu, cực kỳ (Vd: 超高速 - siêu tốc độ, 超能力 - siêu năng lực)
 │   ├─ 大 (Đại): Quy mô lớn, mức độ rất (Vd: 大規模 - quy mô lớn, 大歓迎 - nhiệt liệt chào mừng)
 │   ├─ 高 (Cao): Cao (Vd: 高収入 - thu nhập cao, 高学歴 - học vấn cao)
 │   ├─ 最 (Tối): Nhất, cực đại (Vd: 最高級 - thượng hạng, 最先端 - mới nhất/tốt nhất)
 │   ├─ 低 (Đê): Thấp (Vd: 低価格 - giá thấp, 低気圧 - khí áp thấp)
 │   ├─ 薄 (Bạc): Có chút, hơi (Vd: 薄暗い - hơi tối tăm)
 │   ├─ 小 (Tiểu): Một chút, nhỏ (Vd: 小規模 - quy mô nhỏ, 小論文 - tiểu luận)
 │   └─ 真 (Chân): Nhấn mạnh mức độ tuyệt đối (Vd: 真新しい - mới tinh, 真夜中 - nửa đêm)
 │ ├─ Thời gian
 │   ├─ 新 (Tân): Mới (Vd: 新商品 - sản phẩm mới, 新生活 - cuộc sống mới)
 │   ├─ 旧 (Cựu): Cũ, cựu (Vd: 旧正月 - Tết âm lịch, 旧制度 - chế độ cũ)
 │   ├─ 元 (Nguyên): Trước đây, cựu (Vd: 元首相 - cựu thủ tướng)
 │   ├─ 現 (Hiện): Hiện tại (Vd: 現時点 - thời điểm hiện tại, 現住所 - chỗ ở hiện tại)
 │   ├─ 前 (Tiền): Trước (Vd: 前社長 - giám đốc trước)
 │   ├─ 昨 (Tạc): Trước (Vd: 昨十五日 - 15 ngày trước)
 │   ├─ 今 (Kim): Lần này, mùa này (Vd: 今シーズン - mùa này)
 │   └─ 来/翌/明 (Lai/Dực/Minh): Tiếp theo, tới (Vd: 来学期 - học kỳ tới, 翌朝 - sáng hôm sau)
 │ ├─ Số lượng
 │   ├─ 一 (Nhất): Một (Vd: 一種 - một loại, 一握り - một chút/một nắm)
 │   ├─ 多 (Đa): Nhiều, đa (Vd: 多機能 - nhiều tính năng, 多目的 - đa năng)
 │   ├─ 全 (Toàn): Tất cả, toàn bộ (Vd: 全学生 - tất cả học sinh, 全自動 - hoàn toàn tự động)
 │   ├─ 各 (Các): Mỗi, các (Vd: 各家庭 - mỗi gia đình)
 │   ├─ 諸 (Chư): Các, những (Vd: 諸外国 - các nước khác, 諸事情 - nhiều lý do khác nhau)
 │   └─ 数 (Số): Một vài (Vd: 数時間 - vài giờ)
 │ └─ Lặp lại └─ 再 (Tái): Tái, làm lại (Vd: 再利用 - tái sử dụng, 再開発 - tái phát triển)
